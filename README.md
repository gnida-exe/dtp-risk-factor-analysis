# Анализ факторов риска дорожно-транспортных происшествий

В репозитории представлен алгоритм анализа дорожно-транспортных происшествий,
ориентированный на выявление изменений структуры ДТП и оценку связи отдельных
факторов риска с тяжестью последствий.

## Используемые данные
Для анализа применяются открытые статистические данные о дорожно-транспортных
происшествиях, представленные в формате GeoJSON. В рамках алгоритма выполняется
их преобразование в табличный вид с последующей обработкой.

## Функциональные возможности
Реализованный алгоритм позволяет:
- выполнять предварительную обработку данных;
- формировать показатели структуры ДТП по степени тяжести;
- анализировать влияние условий освещённости и состояния дороги;
- визуализировать результаты анализа в графической форме.

## Использование
Для применения алгоритма требуется загрузить набор данных о ДТП и выполнить
ячейки ноутбука последовательно сверху вниз.

## Назначение
Разработанный подход может быть использован в учебных и исследовательских целях
при анализе статистических данных в области безопасности дорожного движения.
